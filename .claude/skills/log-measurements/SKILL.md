# Запись обмеров тела

Добавь одну строку в `data/measurements.jsonl`.

## Шаги

1. Определи, какие из трёх обмеров названы: талия (`waist_cm`), бёдра
   (`hips_cm`), грудь (`chest_cm`) — хотя бы один должен быть. Мерить не
   обязательно всё за раз. Если непонятна дата — считай сегодняшней. Если
   значение вне реалистичного диапазона (40–200 см) — переспроси.
2. **Допиши запись хелпером** (не правь файл руками):
   ```bash
   python3 scripts/jsonl-edit.py append data/measurements.jsonl '{"date":"2026-07-22","waist_cm":78,"hips_cm":98,"notes":"утром"}'
   ```
   Указывай только реально названные поля. `notes` опционально.
3. Если на ту же дату уже есть запись — спроси, заменить или добавить вторую.
   Замена: `python3 scripts/jsonl-edit.py replace data/measurements.jsonl "<дата>" '<json>'`
   (при двух записях на дату хелпер попросит уточнить `line:N`).
4. Коротко подтверди и, если есть предыдущие обмеры, покажи изменение по
   каждому изменившемуся полю («талия −1 см с прошлого раза»).

> Для графиков динамики обмеров — скилл `view-trends`.
