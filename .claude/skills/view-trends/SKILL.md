# Графики трендов

Показать динамику калорий и веса Unicode-графиками. Только визуализация — без
анализа и советов (для них есть `nutrition-review` и агент `nutrition-coach`).

## Шаг 1. Определи период

Из запроса: «неделя», «месяц», «за июнь», конкретные даты. Если период не
указан — **последние 30 дней** (умолчание скрипта).

Переведи нижнюю границу в дату `YYYY-MM-DD`:
- «неделя» → 7 дней назад; «месяц» → 30 дней; «июнь» → `2026-06-01`;
- «за всё время» → дата первой записи дневника (или заведомо ранняя, `2020-01-01`).

## Шаг 2. Запусти рендерер (без API)

```bash
python3 scripts/render-trends.py [YYYY-MM-DD]   # без даты — последние 30 дней
```

Скрипт сам читает дневник, вес, обмеры тела и норму (`data/norms.json`),
печатает графики, сохраняет их в `data/summary/тренды.md` и копирует в
облачный инбокс (если папка есть) — смотреть с телефона.

## Шаг 3. Покажи результат

Выведи stdout скрипта пользователю как есть (моноширинные графики уже готовы).
Скажи, что копия лежит в `data/summary/тренды.md` и (если скопировалась) в
облачном инбоксе под именем `тренды.md`.

## Грейсфул

Скрипт сам обрабатывает края — не падай и не дублируй его логику:
- пустые данные → заглушка с подсказкой, что записать;
- < 2 взвешиваний → просьба писать вес раз в 1–3 дня;
- < 2 обмеров одного вида (талия/бёдра/грудь) → для него тренд не
  показывается, только подсказка мерить раз в 1–2 недели;
- нет нормы → графики без метки нормы + предложение `setup-profile`;
- битые JSONL-строки пропускаются с предупреждением в stderr.

Если python3 недоступен (крайний случай) — собери короткую текстовую сводку
сам по `data/diary.jsonl` и `data/weight.jsonl`: суммы ккал по дням и дельта веса.

## Как выглядит (пример)

```
## Калории по дням (норма 1480 ккал)
05.07 ████████████                 1486 ⚠
06.07 ██████████████               1728 ⚠
07.07 ████████████                 1453
           ▲ норма
в среднем 1556 ккал/день за 3 дн. с записями (+76 к норме)

## Вес
70 → 71 кг (+1.0) за 12 дн. · 3 взвешиваний
▁█▇   (min 70 · max 71.25)

## Обмеры
Талия: 78 → 76 см (-2.0) за 14 дн. · █▄▁
```
