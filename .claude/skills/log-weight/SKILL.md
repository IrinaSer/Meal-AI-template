# Запись веса

Добавь одну строку в `data/weight.jsonl`.

## Шаги

1. Определи вес в кг из сообщения. Если непонятна дата — считай сегодняшней
   (используй текущую дату). Если непонятны единицы — уточни.
   Если значение вне реалистичного диапазона (20–100 кг) — переспроси.
2. **Допиши запись хелпером** (не правь файл руками):
   ```bash
   python3 scripts/jsonl-edit.py append data/weight.jsonl '{"date":"2026-06-23","weight_kg":68.5,"notes":"утром натощак"}'
   ```
   `notes` опционально (время взвешивания, контекст).
3. Если на ту же дату уже есть запись — спроси, заменить или добавить вторую.
   Замена: `python3 scripts/jsonl-edit.py replace data/weight.jsonl "<дата>" '<json>'`
   (при двух записях на дату хелпер попросит уточнить `line:N`).
4. Коротко подтверди и, если есть предыдущие взвешивания, покажи изменение
   («−0.4 кг с прошлого раза»).

> Для полноценного анализа динамики веса и связи с калоражем — скилл
> `nutrition-review`. Для обмеров тела (талия/бёдра/грудь) — скилл
> `log-measurements`.
