# 🍽️ Meal-AI — твой AI-нутрициолог в Claude Code

Фотографируешь еду → система распознаёт блюдо, считает калории и БЖУ, ведёт
дневник питания, следит за весом и подсказывает, как питаться сбалансированнее.
Всё работает прямо в Claude Code, без приложений, серверов и аккаунтов — только
текстовые файлы у тебя на диске.

> ⚠️ Оценка калорий по фото — **приблизительная**: точную граммовку и количество
> масла/соусов по картинке не узнать. Система всегда показывает уровень уверенности
> и даёт поправить порции. Это помощник для осознанности, а не медицинский прибор.

## 🚀 Быстрый старт

1. **Настрой профиль** — скажи Claude:
   > «Настрой мой профиль питания»

   Он спросит пол, возраст, рост, вес, активность и цель, посчитает твою норму
   калорий и БЖУ и сохранит в `profile.md`.

2. **Записывай еду** — пришли фото тарелки (или этикетки) и скажи:
   > «Запиши, что я съела»

   Claude распознает блюдо, оценит калории/БЖУ, добавит в дневник и покажет,
   сколько калорий осталось на сегодня.

3. **Смотри анализ** — в любой момент:
   > «Как мой день?» / «Отчёт за неделю» / «Проанализируй моё питание»

## 🗣️ Что можно говорить (скиллы)

| Хочешь | Скажи примерно так |
|--------|--------------------|
| Настроить/обновить профиль | «настрой профиль», «я теперь вешу 65, пересчитай норму» |
| Записать приём пищи | «запиши что я съела» (+ фото), «добавь обед» |
| Записать вес | «мой вес сегодня 68», «запиши вес 67.5» |
| Получить сводку и советы | «как мой день», «отчёт за неделю», «анализ питания» |
| Поправить/удалить запись | «удали последний приём», «исправь обед — было 200 г» |
| Почистить старые фото | «почисти старые фото» |
| Разобрать инбокс с телефона | «разбери инбокс», «обработай входящие», «что там с телефона» |

> Точные слова не важны — Claude поймёт по смыслу.

## 📸 Что происходит с фото

Фото нужно только для распознавания — после записи в дневник попадают уже
**посчитанные цифры**. Сами фото хранятся `30 дней` (настраивается в `CLAUDE.md`,
параметр `PHOTO_RETENTION_DAYS`), затем их можно удалить командой «почисти старые
фото». Цифры в дневнике при этом остаются навсегда.

## 💾 Сохранность данных

Проект под `git` — каждое изменение дневника версионируется, можно откатить
ошибочную правку. Папка `photos/` в `.gitignore` (тяжёлая и временная), а твои
данные (`data/`, `profile.md`, `reference/`) — под контролем версий.

## 📱 Логирование с телефона (облачный инбокс)

Не за Маком, но хочешь записать еду? Кидай фото через «Поделиться» в облачную папку
`Meal-AI-Inbox/photos/` (Яндекс.Диск), а текстовые заметки — в `Meal-AI-Inbox/notes.md`.
Клиент Яндекс.Диска синхронизирует это на Мак.

- Дома (или когда угодно) скажи **«разбери инбокс»** — Claude распознает фото и
  заметки, занесёт всё в дневник, перенесёт фото в `photos/YYYY-MM/` и очистит инбокс.
- После разбора обновляется `Meal-AI-Inbox/ответ.md` с остатком калорий/БЖУ за день —
  открываешь его в приложении Яндекс.Диска на телефоне.
- **Авто-режим (Фаза B):** launchd-агент `com.meal-ai.inbox` раз в ~15 минут сам
  запускает разбор (`scripts/process-inbox.sh`) — остаток за день обновляется почти
  в реальном времени, без сервера и API-ключа.

## ✨ Приятные мелочи

- **Живой остаток калорий** — в подвале Claude Code всегда видно
  `🥗 1083/1480 ккал · Б45 Ж22 У90 · ⚡2 приёма` (статус-строка, `scripts/statusline.sh`).
- **Приветствие при старте** — дата, сводка за сегодня, серия дней 🔥 и напоминание
  взвеситься, если давно не писала (`scripts/session-welcome.sh`).
- **Еженедельный авто-дайджест** — launchd-агент `com.meal-ai.weekly` по воскресеньям
  складывает сводку за неделю в `data/summary/ГГГГ-Wнеделя.md` (`scripts/weekly-digest.sh`).

## 🏖️ Режим отпуска (без ноутбука)

Уезжаешь без Мака? Считать калории всё равно можно:
- **Живой счёт** — приложение Claude на телефоне: вставь «карточку отпуска»
  (`Meal-AI-Inbox/Отпуск-режим.md` в облаке) в новый чат и шли туда фото еды —
  Claude ведёт остаток за день. Без Мака и без API-ключа.
- **Официальный учёт** — те же фото сохраняй в `Meal-AI-Inbox/photos/`. Дома скажи
  «разбери инбокс» — весь отпуск занесётся в дневник.

Подробнее: `reference/vacation-mode.md`.
