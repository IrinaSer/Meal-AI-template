# Методичка по оценке порций (для распознавания еды по фото)

Используй это при оценке граммовки и калорийности по фотографии. Цель — разумная
оценка, не идеальная точность. Всегда отмечай уровень уверенности.

## Ориентиры по размеру (визуальные аналогии)

| Аналог | Объём/вес | Типичные продукты |
|--------|-----------|-------------------|
| Кулак / теннисный мяч | ~150 г | порция каши, макарон, риса (готовых) |
| Ладонь без пальцев | ~100–120 г | кусок мяса/рыбы/птицы |
| Сложенная горсть | ~30 г | орехи, сухофрукты, чипсы |
| Большой палец | ~15 г / 1 ст. л. | масло, ореховая паста, майонез |
| Кончик большого пальца | ~5 г / 1 ч. л. | масло, сахар, мёд |
| Колода карт | ~85 г | мясо, сыр |
| Стандартная тарелка ⌀26 см | — | полная тарелка вторго ≈ 350–450 г |

## Скрытые калории (часто недооцениваются)

- **Масло для жарки:** +100–150 ккал на порцию (1–1.5 ст. л. впитывается).
- **Заправка салата:** соус/масло легко добавляет +100–200 ккал.
- **Соусы:** майонез ~700 ккал/100 г, сметана ~200, кетчуп ~100, песто ~450.
- **Сахар в напитках/выпечке:** 1 ч. л. сахара ≈ 20 ккал.
- **Панировка, кляр:** +20–40% к калорийности куска.

## Как формулировать уровень уверенности

- **high** — этикетка с составом, или простое моноблюдо известной граммовки.
- **medium** — узнаваемое блюдо, порция оценена визуально.
- **low** — сложное/смешанное блюдо, непонятный способ готовки, плохое фото.
  Дай диапазон и попроси пользователя уточнить.
